### ⭐️ 最優先: Releaseビルドの動作確認
**理由**: デバッグログ無効化や最適化設定が正しく動作するか確認が必須

**Windows**:
```
Win/OST_WindyLines.sln を開く
→ Release/x64 に切り替え
→ ビルド実行
→ Premiere Pro で動作確認
```

**macOS**:
```
Mac/OST_WindyLines.xcodeproj を開く
→ Release スキームに切り替え
→ ビルド実行
→ Premiere Pro で動作確認
```

**確認項目**:
- ✓ ログファイルが生成されないこと
- ✓ デバッグマーカーが表示されないこと
- ✓ パフォーマンスがDebugより高速なこと
- ✓ すべての機能が正常動作すること

**所要時間**: 30分-1時間
