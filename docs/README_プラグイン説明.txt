おしゃれテロッププラグイン（風を感じるライン）

このプラグインは、映像上に「流れる線」の演出を追加するためのエフェクトです。
線の生成、色、動き、描画範囲、影、モーションブラー、合成方法を調整できます。

==================================================
GPUに関する注意（重要）
==================================================
- 本プラグインは GPU レンダリング対応です（環境により CPU レンダリングになる場合があります）。
- 初回適用直後は、GPU側の準備でプレビューが一時的に重くなることがあります。
- GPUドライバやOS更新直後に表示が不安定な場合は、Premiere Pro の再起動をお試しください。
- 環境差（GPU/ドライバ/OS）により、見た目や処理速度にわずかな差が出る場合があります。

==================================================
主な使い方
==================================================
1) エフェクトをクリップに適用
2) 「プリセットパラメーター」で大まかな見た目を選択
3) 下記パラメータで線の量・色・動き・範囲を調整
4) 必要に応じて「ライセンス > アクティベート」を実行

==================================================
パラメータ一覧（全項目）
※ カラープリセットの各プリセット名は省略
==================================================
[トップ]
- プリセットパラメーター：全体の見た目をまとめて呼び出します。
- ランダムシード：線の出方の乱数パターンを切り替えます。

[基本]
- 線の数：同時に表示される線の本数を調整します。
- 寿命(fps)：1本の線が表示される時間を設定します。
- インターバル(fps)：新しい線を発生させる間隔を設定します。

[カラー]
- カラープリセット（単色 / カスタム / プリセット）：色の指定方式を選びます。
- 単色カラー：単色モード時に使う色を指定します。
- カスタムカラー：カスタムパレットの色を指定します。
- イージング：線の進行速度カーブ（加減速）を選びます。
- 移動距離の連動：移動距離を範囲サイズに連動させるか選びます。
- 移動距離連動率(%)：連動時の反映割合を調整します。
- 移動距離(px)：線が移動する距離の基準値を設定します。

[見た目]
- 太さの連動：線の太さを範囲サイズに連動させるか選びます。
- 太さ連動率(%)：太さ連動の反映割合を調整します。
- 太さ(px)：線の太さを設定します。
- 長さの連動：線の長さを範囲サイズに連動させるか選びます。
- 長さ連動率(%)：長さ連動の反映割合を調整します。
- 長さ(px)：線の長さを設定します。
- 線の角度：線の基本角度を設定します。
- シアー（傾斜）：線形状を平行四辺形方向に傾けます。
- 端のスタイル：線の先端形状（フラット/ラウンド）を選びます。
- テールフェード：線の後端側のフェード量を調整します。

[起点と描画範囲]
- 範囲ソース：発生範囲を画面全体/元要素から選びます。
- 元要素のアルファしきい値：元要素範囲判定に使う透明度しきい値です。
- 起点モード：線の発生基準位置の考え方を選びます。
- 進行方向：線の進み方パターンを選びます。
- 開始時間(fps)：線アニメーションの開始タイミングを設定します。
- 継続時間(fps)：線アニメーションの継続時間を設定します。0を指定するとエンドレスとなります。
- 奥行き強度：前後感（奥行き演出）の強さを調整します。
- 中央ギャップ：中央付近に空ける余白の量を設定します。
- 範囲オフセットX(px)：発生範囲を水平方向に移動します。
- 範囲オフセットY(px)：発生範囲を垂直方向に移動します。
- 範囲スケールX(%)：発生範囲の横幅倍率を調整します。
- 範囲スケールY(%)：発生範囲の縦幅倍率を調整します。
- 範囲の回転：発生範囲全体を回転します。
- 目安の表示：発生範囲ガイドの表示/非表示を切り替えます。
- 目安の色：発生範囲ガイドの色を指定します。

[シャドウ]
- 影を有効化：線の影の表示をオン/オフします。
- 影色：影の色を設定します。
- オフセットX：影の横方向ずれ量を設定します。
- オフセットY：影の縦方向ずれ量を設定します。
- 不透明度：影の濃さを設定します。

[モーションブラー]
- モーションブラー：移動残像の有効/無効を切り替えます。
- サンプル数：残像計算の品質（分割数）を調整します。
- シャッター角度：残像の長さ・強さの基準を調整します。

[その他の設定]
- アンチエイリアス：線の縁のなめらかさを調整します。
- 要素を隠す：元の要素を非表示にして線のみ表示します。
- 合成モード：線と元要素の重なり方を選びます。

[ライセンス]
- アクティベート：ライセンス認証を実行します。

==================================================
補足
==================================================
- パラメータが多い場合は、まず「プリセットパラメーター」で方向性を決めてから微調整すると効率的です。
- 「要素を隠す」を有効にすると、元の映像要素を非表示にして線のみ表示できます。
