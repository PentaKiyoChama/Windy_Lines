OST WindyLines v1.0 - 風を感じるラインエフェクト
for Adobe Premiere Pro

========================================
製品概要
========================================

パーティクルベースの流線エフェクトプラグイン。
映像に沿って流れる美しい線を生成し、動画をアート作品に変換します。

【主な機能】

◆ 高度なパーティクルシステム
  - 最大5000本の同時レンダリング
  - リアルタイムGPUアクセラレーション (CUDA/Metal/OpenCL/DirectX)
  - 28種類のイージング関数

◆ 豊富なカスタマイズ
  - 50種類以上のプリセット
  - カスタムカラーパレット（8色）
  - 線の太さ・長さ・速度の詳細調整
  - シアー変形（平行四辺形）
  - 影・モーションブラー対応

◆ プロフェッショナル品質
  - 4K/8K解像度対応
  - アンチエイリアシング
  - アルファチャンネル完全対応

========================================
システム要件
========================================

【Windows】
- OS: Windows 10/11 (64bit)
- CPU: Intel Core i5 以上
- GPU: NVIDIA GTX 1060 以上 (推奨)
- RAM: 8GB以上
- Adobe Premiere Pro: CC 2018 以降

【macOS】
- OS: macOS 11 (Big Sur) 以降
- CPU: Apple Silicon / Intel Core i5 以上
- GPU: Metal 対応
- RAM: 8GB以上
- Adobe Premiere Pro: CC 2018 以降

========================================
基本的な使い方
========================================

1. Premiere Pro でクリップを選択
2. エフェクトパネルから「流れる線」を適用
3. エフェクトコントロールで調整:
   - エフェクトプリセット: クイック適用
   - 本数: 線の数（推奨: 100-500）
   - 寿命: 表示時間（フレーム数）
   - 移動距離: 線が進む距離

4. カラー設定:
   - 単色: 1色で統一
   - プリセット: 50種類以上のカラーパレット
   - カスタム: 8色まで自由に設定

5. レンダリング:
   - シーケンス設定でGPUアクセラレーションを有効化
   - 複雑な設定では線の本数を減らすと軽快に動作
